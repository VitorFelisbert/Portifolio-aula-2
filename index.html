Quem eu sou?

Sou programador Full Stack, o que significa que atuo no desenvolvimento do back-end e do front-end de aplicações. Tenho conhecimento para criar desde a estrutura lógica e os bancos de dados até as interfaces e experiências do usuário final.

Gosto de entender o projeto como um todo, contribuindo desde a ideia inicial até a entrega final. Essa versatilidade me permite adaptar às necessidades de cada projeto e colaborar de forma completa com qualquer equipe.
